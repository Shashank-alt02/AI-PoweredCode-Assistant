### Prerequisites
```bash
Python 3.8 or higher
pip package manager
OpenAI API key (for GPT integration)
```

### Installation

```bash
# Clone the repository
git clone https://github.com/Shashank-alt02/AI-PoweredCode-Assistant.git

# Navigate to project directory
cd AI-PoweredCode-Assistant

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Add your OpenAI API key to .env file
```

### Usage

```python
from code_assistant import AICodeAssistant

# Initialize the assistant
assistant = AICodeAssistant(api_key="your_api_key")

# Get code suggestions
suggestion = assistant.complete_code("def calculate_fibonacci")

# Detect bugs
bugs = assistant.analyze_code(your_code)

# Refactor code
refactored = assistant.refactor(your_code, style="clean_code")
```
